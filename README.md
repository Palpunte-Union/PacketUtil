## How To Build
 - 基本的にReleaseから落としてください。
 - gradle wrapper入ってるのでビルドしろください
 - あ、でもビルドにはcraftbukkitとかspigotのjarファイルがいります自分でビルドしてきてください

## Requirements
 - Java
 - CraftBukkit or Spigot jar 1.13 ~ 1.16.5
